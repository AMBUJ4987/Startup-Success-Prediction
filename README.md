# Startup Success Prediction

## Overview

Startup Success Prediction is a Machine Learning project designed to analyze historical startup data and predict the likelihood of a startup's success. The project uses data preprocessing, exploratory data analysis (EDA), and predictive modeling techniques to identify patterns and factors associated with successful startups.

The goal of this project is to demonstrate how data-driven insights can support decision-making in the startup ecosystem.

---

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Startup trend visualization
* Analysis of funding and industry impact
* Machine Learning-based prediction model
* Performance evaluation and result interpretation

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Power BI

---

## Dataset

The project uses startup-related historical data containing information such as:

* Industry
* Country
* Funding details
* Startup age
* Investment patterns
* Success indicators

The dataset was cleaned and processed before being used for analysis and model training.
